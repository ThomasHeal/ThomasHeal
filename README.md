### About Me  
I am a software developer looking to expand the growing field of technology and software as a service. With my experience developing games as well as client-facing applications and websites I have learned to create efficient and quality services that provide enjoyment as well as improvement of quality of life for end users. 

I specialize in designing efficient backend services that take the code out of the end users hand and replaces it with an appealing front end service that puts them in control. I enjoy creating meaningful experiences that provide a useful and functional tool for day to day tasks.
